📦 1. 什么是数组 (Array)？

在计算机科学中，数组 (Array) 就像是一个定制的鸡蛋盒或药盒。

它有几个相对“死板”的规定：

  • 👉 只能装同一种东西：比如全装数字，或者全装字母，不能混搭。
  • 👉 格子数量固定：买来是10个格子，就只能装10个，不能临时变大。

虽然死板，但它的好处是找东西极快，而且非常省空间，适合处理大量单一类型的数据（比如图像的像素、大量统计数据）。

array_concept.py
# Python 中如果非要用严格的数组，需要引入专门的模块
from array import array

# 'i' 代表整数(integer)，规定这个数组只能装整数
strict_array = array('i', [1, 2, 3])
print("严格的数组:", strict_array)

# 如果强行塞入一个文字，就会报错！
# strict_array.append("苹果") # ❌ TypeError
