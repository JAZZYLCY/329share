🔄 5. for 循环里的下划线

核心定义

语法层面：_ 是一个合法的变量名，完全符合 Python 语法规则。

社区约定：在 Python 开发社区中，_ 被赋予了特定的语义——它专门用来表示 “我不需要这个值”（I don't care about this value）。

典型应用场景

场景 代码示例 含义解析
循环次数 for _ in range(5): 只需执行 5 次循环逻辑，不关心每次循环产生的计数器值
解包忽略 a, _, c = (1, 2, 3) 解构元组/列表时，忽略中间的那个元素

为什么使用 _？

可读性提升：明确地向阅读代码的人传递信号：这个变量是占位用的，后续逻辑不会使用它。

避免警告：在代码编辑器中，如果定义了变量却未使用，通常会触发 linter 的警告。使用 _ 可以明确规避这种警告。

loop_demo.py
# 示例 1：循环次数
print("--- 循环次数示例 ---")
# 循环 5 次并打印 "Hello"，无需使用循环变量
for _ in range(5):
    print("Hello")

# 示例 2：解包忽略
print("\n--- 解包忽略示例 ---")
# 解构元组，只保留第一个和第三个元素，忽略中间的 "banana"
first, _, last = ("apple", "banana", "cherry")
print(first, last)

Python 变量名命名规则

Python 变量名必须遵守 3 条铁律（不遵守就是非法）

1. 只能用：字母、数字、下划线

可以：name、my_var、user123

不行：user@name、my-var、hello!

2. 不能以数字开头

可以：a1、_123

不行：123a、1name

3. 不能是 Python 自带的关键字

比如：if、for、else、True、False、def、class

你不能写：if = 10 → 直接报错！
