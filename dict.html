📖 3. 字典 {}

列表是用顺序（位置）来找东西的，而 字典 (Dictionary) 是用名字来找东西的！

它使用大括号 {}，里面存的是键值对 (Key-Value)。

就像查现实中的字典：你通过“词语（键）”去查找它的“解释（值）”。非常适合存储有标签的数据，比如学生信息。

dict_demo.py
# 使用 {} 创建字典："键": "值"
student = {
    "name": "小明",
    "age": 18,
    "course": "Python"
}

# 通过键（标签）来获取对应的值
print("学生姓名是:", student["name"])

# 随时添加新标签
student["score"] = 99
print("更新后的字典:", student)
