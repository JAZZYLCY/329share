🔢 3. 索引：怎么从列表里拿东西

列表里放了那么多东西，我怎么拿出其中一个呢？用 索引！

📝 索引示意图

索引: 0 1 2 3
内容: "牛奶" "面包" "苹果" "鸡蛋"
反向索引: -4 -3 -2 -1
  • ✨ 重要：从 0 开始数！ 第1个是 [0]，第2个是 [1]...
  • ✨ 用负数可以从尾巴倒着数：[-1] 是最后1个，[-2] 是倒数第2个...
index_demo.py
# 购物车列表
cart = ["牛奶", "面包", "苹果", "鸡蛋"]

print("购物车:", cart)

# --- 正向索引 ---
print("第1件商品:", cart[0])   # 0 是第1个
print("第3件商品:", cart[2])   # 2 是第3个

# --- 反向索引 ---
print("最后1件:", cart[-1])    # -1 是最后1个
print("倒数第2个:", cart[-2])  # -2 是倒数第2个

⚠️ 易错点提醒

  • ❌ 索引越界：访问不存在的索引（如长度为4的列表访问 cart[4]）会直接报错
