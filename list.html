🎒 2. 列表 (List)

列表是 Python 中最常用的数据结构之一！

它就像一个 "万能魔法背包"：

  • 👉 什么都能装：数字、文字、布尔值、甚至另一个列表！
  • 👉 随意变大变小：随时追加新东西，背包自动适应大小。

在 Python 中，我们用方括号 [] 来创建列表。

list_demo.py
# 创建一个列表
my_list = [100, "苹果", True, 3.14]

print("初始背包:", my_list)

# 往列表里追加新东西
my_list.append("新朋友")

print("装入新东西后:", my_list)
