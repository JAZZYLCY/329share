✏️ 4. 列表常用方法详解

学会了怎么拿东西，现在来学更丰富的操作！

list_methods.py
# 定义列表 fruits
fruits = ["apple", "banana", "cherry"]
print("1. 初始列表:", fruits)

# 2. 按索引访问元素（索引从0开始）
print("\n2. 第一个水果:", fruits[0])
# fruits[0] 对应第一个元素，fruits[1]是banana，fruits[2]是cherry

# 3. append()：在列表末尾添加元素
fruits.append("orange")
print("\n3. append后:", fruits)

# 4. insert()：在指定索引位置插入元素
fruits.insert(1, "grape")
print("\n4. insert(1, 'grape')后:", fruits)
# 在索引1的位置插入grape，原索引1及之后的元素后移

# 5. pop()：弹出（删除并返回）元素，默认弹出最后一个
popped = fruits.pop()
print("\n5. pop()弹出的:", popped)
print("   pop后:", fruits)
# 弹出最后一个元素"orange"，popped变量值为"orange"

# 6. len()：获取列表长度（元素个数）
print("\n6. 列表长度:", len(fruits))

📋 常用列表方法速查表

方法 语法 功能说明 示例
append(x) 列表.append(元素) 在列表末尾添加一个元素 fruits.append("orange")
insert(i, x) 列表.insert(索引, 元素) 在指定索引位置插入元素，原元素后移 fruits.insert(1, "grape")
pop([i]) 列表.pop(可选索引) 删除并返回指定索引的元素，不填索引则删除最后一个 fruits.pop() / fruits.pop(1)

⚠️ 易错点提醒

  • ❌ 索引越界：访问不存在的索引（如长度为4的列表访问 fruits[4]）会直接报错
  • ❌ pop(i)：索引越界同样会报错
