📊 两种"数组"的对比

Python 中提到的"数组"其实有两个概念，表格对比最清晰：

特性 列表 (List) array 模块 (类型化数组)
语法符号 直接使用 [] 需要导入模块：from array import array
数据类型 超级灵活：可以混合存储
例：[1, "hello", True]		 严格限定：只能存同一种类型
例：只能存整数，或只能存浮点数
内存占用 较大（存的是指针，支持任意类型） 紧凑（存的是原始二进制数据）
适用场景 绝大多数业务场景 高性能数值计算、科学运算、底层数据处理

💻 代码逻辑解析

comparison.py
# --- 1. 列表作为动态数组（日常最常用）---
print("=== 列表演示 ===")
# 定义一个列表，混合了整数和字符串
arr = [1, 2, 3, "hello"]
print("初始列表:", arr)

# append(4)：动态添加元素
arr.append(4)
print("append后:", arr)
# ✅ 特点：自动扩容，类型不限，随意添加

print("\n=== array 模块演示 ===")
# --- 2. 使用 array 模块（进阶性能向）---
# 第一步：导入特定的 array 类
from array import array

# 第二步：创建严格类型的数组
# 'i' 表示规定：这里面只能存 integer（整数）
int_arr = array('i', [1, 2, 3])
print("严格数组:", int_arr)

# ✅ 正常添加
int_arr.append(4)
print("append(4)后:", int_arr)

# ❌ 报错！因为规定了只能存数字
# int_arr.append("hello")  # 这行会报错！

💡 类型码说明：
array('i', ...)：'i' 是类型码，代表整型。
常见的还有 'f' 代表浮点型，'d' 代表双精度浮点。
