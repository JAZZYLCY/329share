✏️ 4. 常用字典方法速查表

方法 功能说明 示例
keys() 获取字典中所有的键，返回可迭代对象 person.keys() → dict_keys(['name', 'city'])
values() 获取字典中所有的值，返回可迭代对象 person.values() → dict_values(['Alice', 'Beijing'])
items() 获取字典中所有的键值对，返回可迭代对象 person.items() → dict_items([('name', 'Alice'), ('city', 'Beijing')])
get(key) 安全获取键对应的值，键不存在时返回默认值（不报错） person.get("age", 0) → 0
pop(key) 删除指定键的键值对，并返回对应的值 person.pop("name") → "Alice"
dict_methods.py
# 定义一个字典
person = {"name": "Alice", "city": "Beijing", "age": 25}
print("初始字典:", person)

# keys()：获取所有键
print("\n所有键:", list(person.keys()))

# values()：获取所有值
print("所有值:", list(person.values()))

# items()：获取所有键值对
print("所有键值对:", list(person.items()))

# get()：安全获取值
print("\n获取 age:", person.get("age", 0))
print("获取不存在的键:", person.get("phone", "未提供"))

# pop()：删除键值对
popped = person.pop("age")
print("\n删除 age:", popped)
print("删除后:", person)
