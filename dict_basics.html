📖 3. 字典（dict）核心特性

核心特点

  • 通过唯一的键快速查找值
  • 键必须是不可变类型
  • 支持增、删、改、查

什么叫键值对？

键值对 = 钥匙 + 锁起来的东西

  • 键（key）= 钥匙
  • 值（value）= 箱子里的东西
dict_demo.py
# 定义字典 person
person = {"name": "Alice", "age": 25}
print("1. 初始字典:", person)

# 2. 通过键访问
print("\n2. 姓名:", person["name"])

# 3. 新增键值对
person["city"] = "Beijing"
print("\n3. 新增后:", person)

# 4. 删除键值对
del person["age"]
print("\n4. 删除后:", person)

# 5. 遍历
print("\n5. 遍历:")
for k, v in person.items():
    print(f"   {k}: {v}")
