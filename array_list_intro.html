📦 1. Python 数组有两种

一、内置列表 list（动态数组）

Python 最常用的序列类型，用 [] 定义。

  • 可存任意类型混合数据
  • 索引从 0 开始，有序排列
  • 支持增、删、改、查

二、array 模块（类型化数组）

Python 标准库提供的专用数组，需要导入。

  • 只能存同一种类型的数据
  • 内存存储紧凑，性能更好
  • 适合数值计算
array_concept.py
# 列表 list（动态数组）
my_list = ["apple", 123, True]
print("列表:", my_list)

# array 模块（类型化数组）
from array import array

# 'i' 代表整数，只能存整数
num_array = array('i', [1, 2, 3])
print("类型化数组:", num_array)
